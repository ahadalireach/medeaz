# Medeaz Healthcare Platform

Voice-enabled healthcare platform connecting doctors, patients, and clinics.

## Core Tech Stack
- **Frontend**: Next.js 16.1.6, React 19, TailwindCSS 4, Redux Toolkit, Next-intl
- **Backend**: Node.js (Express 5), MongoDB (Mongoose 9), Socket.io
- **Key Services**: Google Generative AI, Cloudinary, Node-cron

## Development Status
Current focus is on bug fixes, UI polish, and communication enhancements.

## Core Pillars
1. **Records Management**: Digital and voice-generated prescriptions.
2. **Communication**: Real-time chat and notifications.
3. **Analytics**: Revenue and flow tracking for clinics and doctors.
4. **Localization**: English and Urdu support.
